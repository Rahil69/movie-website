# Movie Website

A React + Vite movie search app that lets users browse popular movies and search for specific titles using The Movie Database API.

## Features

- Browse popular movies
- Search movies by title
- Debounced search to reduce API requests
- Loading spinner while fetching results
- Error message handling
- Responsive movie cards
- Styled with Tailwind CSS

## Tech Stack

- React
- Vite
- Tailwind CSS
- TMDB API
